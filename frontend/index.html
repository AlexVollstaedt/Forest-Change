Forest Change Detection App | Select any area on earth to detect deforestation patterns over time

About this App

The Forest Change Detection App is an interactive web application that analyzes forest cover changes using satellite imagery from Google Earth Engine.

How to use:

  • Search for a location or navigate the map manually
  • Adjust the time period slider to select your analysis timeframe (2000-2024)
  • Click "Select Area on Map" to draw a rectangle over your area of interest
  • Click "Analyze Forest Change" to process the data
  • Click "Download Yearly Data" to get the processed data in .csv format

Data Source: This application uses the Hansen Global Forest Change dataset (v1.12, 2024) accessed through Google Earth Engine API.

Basemap: The basemap used for this application is the ESRI World Imagery basemap in combination with ESRI World Boundaries and Places labels

Purpose: This is a personal portfolio project created to demonstrate geospatial analysis and web development skills.
